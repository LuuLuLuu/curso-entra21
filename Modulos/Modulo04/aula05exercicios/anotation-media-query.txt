* Sobre media query

- Uso:
	Apenas para criar um design responsivo.

"Em suma é um meio de aplicar uma estilização quando o navegador ou o dispositivo tiver uma certa regra ou condição."

- Para especificar uma regra :
	(num arquivo .css)
	@media __tipo__ (expressão) {estilização}

	Ou seja, tu define que é uma media query(@media) dai tu indica o tipo/type() dai se nescessário uma expressão e por fim a estilização.
